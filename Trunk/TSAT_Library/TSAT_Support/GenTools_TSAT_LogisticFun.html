TSAT: Logistic Function Block

This block computes the logistic function given x as an input. The exact shape of the logistic function is determined based on it's parameters.

$$ y = \frac{L_f - L_o}{1 + e^{-k \left( x - x_o \right) }} + L_o $$

where

$$ L_o = the \: value \: of \: the \: function \: at \: the \: left \: asymptote $$ $$ L_f = the \: value \: of \: the \: function \: at \: the \: right \: asymptote $$ $$ x_o = the \: location \: of \: the \: inflection \: point \: (center) \: of \: the \: curve $$ $$ k = the \: slope \: of \: the \: curve $$

All of these parameters can be specified and some of them can be calculated apriori by the block given some additional information. In particular xo and k can be computed by the block. $$ x_o = 0.5 \left(x_L + x_R\right) $$ $$ k = -\frac{ln(99)}{x_L-x_o} $$

where

$$ x_L = location \: where \: y \: is \: within \: 1 \: percent \: of \: L_o $$ $$ x_R = location \: where \: y \: is \: within \: 1 \: percent \: of \: L_f $$ Logistic Function Inputs:
Logistic Function InputsDescription
xindependent variable [nx1]

Logistic Function Outputs:
Logistic Function OutputsDescription
